What is this demonstrating?

  • Books on the Web can be read offline—on flights, holidays, trains, etc.
  • No special format; it’s just the Web—simply a normal set of Web pages; no separate packaging or archiving is needed for offline use
  • No server trickery needed—publications can be hosted statically
  • No heavy framework required, no render-blocking JavaScript in the way of content—so it is fast and simple to deploy

How does this use Service Workers?

This Web publication uses a Service Worker to intercept all network requests for its constituent resources, and to load them from a local cache rather than over the Internet. The end result for you as a user is a seamless reading experience even when you are offline.
